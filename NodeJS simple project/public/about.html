About This Project

This Node.js demo project was created to show how a simple web server can handle routing and serve static files without using external frameworks like Express.

Technologies Used

This project uses Node.js core modules — http for the server, path for handling file paths, and fs for reading files.

Goal

The goal is to demonstrate how basic routing works in Node.js and how you can serve HTML pages and JSON data from a custom-built server.
