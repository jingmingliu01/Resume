每个元素占1列

示例2: 跨2列 (grid-column: 1/3, 3/5)

第一个元素占第1-2列，第二个元素占第3-4列

1/3
3/5

示例3: 跨3列 (grid-column: 1/4, 4/5)

第一个元素占第1-3列，第二个元素占第4列

1/4
4/5

您代码中的实际应用：

  • grid-column: 1 / 2 - sidebar占第1列
  • grid-column: 3 / 5 - main占第3-4列（2列宽度）
  • grid-column: 2 / 3 - content1占第2列
  • grid-column: 2 / 5 - footer占第2-4列（3列宽度）
