About This Express.js Project

This demo project showcases how to build a simple web server using Express.js — a popular Node.js framework for handling routes, serving static files, and creating API endpoints.

Technologies Used

This project uses:

Express.js — routing & requests
Node.js — JavaScript runtime
path — safe file path handling
fs — saving & reading JSON data

Project Goal

The goal is to demonstrate how Express.js can serve multiple web pages, provide JSON APIs, and dynamically update stored data — all within a clean and lightweight server setup.
